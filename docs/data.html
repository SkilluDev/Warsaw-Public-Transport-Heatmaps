Warsaw Public Transport Data

The data for this project was provided by the public Warsaw API at https://api.um.warszawa.pl/.

More bus facts and data can be found at:

Total Sets of Stops: 2982

Some graphs were generated automatically so they might be less clear than the other ones. The ones I manually created have logarithmic scale versions as well as linear. Instead the automatic ones have a version where 0 is included and is not included.

The hand-picked graphs have additionally an overlapping exponential function that closely resembles the data distribution. This was added because I found the exponential growth very fascinating and I wanted to see how comparable they are. It turned out that the data is very in line with the functions. I suppose that some corelations can be explained by the nature of a crossroad, where if on every intersection you can go either "left" or "right", the lines would diverge in a 8-4/4-2/2/2/2-1/1/1/1/1/1/1/1 manner. However this explanation is still kind of lacking and an approximation, and it might be that this is merely by accident. If someone has an idea about what the source of this familiarity might be, feel free to reach me @ SkilluPlayz.
